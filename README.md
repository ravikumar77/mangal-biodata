# Mangal Biodata

A beautiful, dynamic, and state-driven web application for generating traditional Hindu Marriage Biodatas. 

## Features
- **Dynamic Form Builder:** Add, edit, remove, and reorder biodata fields in real-time.
- **Multiple Themes:** Choose between Classic Red, Elegant Gold, Floral Bloom, Royal Blue, Rose Gold, Minimal Saffron, and Ornate Gold Royal.
- **Religion Icons:** Customizable god icons correctly tinted to match the selected theme.
- **Client-Side Export:** Privately generate perfectly formatted HD PDFs and Images entirely in the browser using `html2pdf.js` and `html2canvas`.
- **Responsive Layout:** Live split-pane preview on desktop and a smooth experience on all devices.

## How to Run Locally
Simply serve the files using any static server.
```bash
npx serve .
```
